# 🌾 KrishiMitr – The Complete Offline Farming Platform

**KrishiMitr** is an AI-powered digital assistant designed to help farmers with real-time agricultural insights, crop guidance, mandi prices, weather forecasts, and pest detection.
This project was developed as part of **Smart India Hackathon (SIH)** to bridge the gap between modern technology and traditional farming.

---

## 🚀 Live Demo

🔗 https://krishi-mitr-5x36.vercel.app/

---

# ✨ Features

## 🤖 AI Advisory Chatbot

* AI-powered chatbot for agriculture queries
* Voice-to-Text and Text-to-Speech support
* Multilingual support (English, Hindi, Punjabi)
* Provides instant crop and farming advice

---

## 🌤 Weather & Mandi Insights

* Hyper-local weather forecast using GPS
* Real-time mandi price tracking
* Helps farmers decide the best time and place to sell crops

---

## 🌱 Soil & Crop Roadmap

* Personalized fertilizer recommendations
* Soil-based crop planning
* Helps farmers optimize crop yield

---

## 🐛 Pest & Disease Detection

* Upload crop or leaf images
* AI analyzes pests and plant diseases
* Suggests treatments and prevention methods

# 📶 Offline First Approach

KrishiMitr is designed for rural areas where internet connectivity may be unreliable.

Key Offline Features:

* **PWA Support** – Installable like a mobile app
* **Caching** – Important data stored locally
* **IndexedDB** – Stores mandi prices and user data
* **Service Workers** – Enable offline functionality
* **Auto Sync** – Updates data when internet becomes available

# ⚙ Installation & Setup

### Clone the repository

```
git clone https://github.com/yourusername/krishi-mitr.git
cd krishi-mitr
```

### Install dependencies

```
npm install
```

### Run the development server

```
npm run dev
```

Visit:

```
http://localhost:3000
```
